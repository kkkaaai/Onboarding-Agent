SLACK ETIQUETTE
---------------

DO:
✓ Use threads to keep conversations organized
✓ Use @channel sparingly (only for urgent, everyone-relevant items)
✓ Set your status when away/busy/in a meeting
✓ Use emoji reactions instead of "thanks" messages
✓ Search before asking (check pinned items and Notion)
✓ Respond within 24 hours during work days

DON'T:
✗ Send "hey" without context (just ask your question)
✗ DM someone who you don't know without intro
✗ @here or @channel in large channels for non-urgent items
✗ Cross-post the same message in multiple channels
✗ Expect instant responses (people may be deep in work)

USEFUL SLACK FEATURES
---------------------

Slash Commands:
/remind - Set reminders
/giphy - Add fun GIFs
/poll - Create quick polls
/zoom - Start a Zoom call

Shortcuts:
Cmd/Ctrl + K - Quick switcher
Cmd/Ctrl + Shift + A - All unreads
Cmd/Ctrl + / - Keyboard shortcuts help

Tips:
- Star important channels for quick access
- Save messages for later with bookmark icon
- Use "Do Not Disturb" for focus time
- Customize your notification settings per channel
