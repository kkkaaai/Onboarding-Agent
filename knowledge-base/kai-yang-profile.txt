THE AURORA DESIGN SYSTEM
------------------------------------
Overview:
Aurora Design System is Raspberry Coffee's next-generation design and component
framework. It delivers a cohesive, accessible, and sensory-rich experience across
hardware and software. Built as a Figma + React component library, it standardizes
color tokens, typography, motion, and layout spacing across all Raspberry products.

GitHub Repository:
https://github.com/JieHan-eng/aurora-design-system

Goals (FY25):
- Unify product visuals across hardware and software.
- Build responsive React components following Bloom OS design rules.
- Integrate automated contrast and keyboard accessibility testing.
- Publish private NPM package for internal teams.
- Build documentation site with Storybook + Docusaurus.
